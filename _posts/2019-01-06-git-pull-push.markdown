# git 入门教程  #
1. 初始化一个空的git仓库
2. 列出详细信息,在每一个名字后面列出其远程url 此时, -v选项(译注:此为 –verbose的简写,取首字母),显示对应的克隆地址
3. 添加远程仓库
4. 拉取远程仓库到本地
5. 他会监控工作区的状态树，使用它会把工作时的所有变化提交到暂存区
6. 提交暂存区的文件
7. 本地分支的更新，推送到远程主机
8. 单独合并一个提交
